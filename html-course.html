1. Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. Dengan HTML, kita bisa menentukan struktur, teks, gambar, link, dan elemen-elemen lain di halaman web.

<!DOCTYPE html>
<html>
  <head>
    <title>Halaman Pertama</title>
  </head>
  <body>
    Halo Dunia!
  </body>
</html>

2. Struktur Dasar HTML

Sebuah halaman HTML memiliki struktur utama:

  • <!DOCTYPE html> ‚Üí deklarasi dokumen HTML5
  • <html> ‚Üí elemen utama
  • <head> ‚Üí informasi halaman (judul, link css, dll)
  • <body> ‚Üí isi halaman
<!DOCTYPE html>
<html>
  <head>
    <title>Contoh Struktur</title>
  </head>
  <body>
    <h1>Judul Halaman</h1>
    <p>Ini paragraf pertama.</p>
  </body>
</html>

3. Heading & Paragraf

HTML menyediakan heading dari <h1> hingga <h6>, dan paragraf dengan <p>.

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<p>Ini adalah sebuah paragraf.</p>

4. Teks & Format Dasar

Kita bisa memberi format teks dengan elemen berikut:

  • <b> ‚Üí tebal
  • <i> ‚Üí miring
  • <u> ‚Üí garis bawah
  • <small> ‚Üí teks kecil
<p>Teks <b>tebal</b>, <i>miring</i>, dan <u>garis bawah</u>.</p>

5. Gambar & Link

Gunakan <img> untuk gambar, dan <a> untuk link.

<img src="https://via.placeholder.com/150" alt="Contoh Gambar">
<p>Kunjungi <a href="https://www.google.com">Google</a></p>
